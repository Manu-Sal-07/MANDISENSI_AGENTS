2. EXACT CHANGES MADE
================================================================================

1. connection.py (Database Pool):
   - Configured `_get_db_url()` to check `APP__ENVIRONMENT`. If set to `production` and `DATABASE_URL` is empty, it raises a `ValueError` immediately rather than silently defaulting to a local development string.

2. client.py (Database Client):
   - Configured the client connection pool setup inside `init()` to fail-fast in a `production` environment if `DATABASE_URL` is not provided.

3. cache.py (Redis Client):
   - Upgraded `_get_redis_client()` host resolver. If running in `production`, it raises a `ValueError` if neither `REDIS_URL` nor `REDIS_HOST` is supplied, instead of falling back to localhost.

4. main.py (FastAPI Server CORS):
   - Replaced static localhost development array with a dynamic origin parser. Allowed origins are now read from the `CORS_ALLOWED_ORIGINS` environment variable (comma-separated list), falling back to local port arrays if the environment variable is unset.

5. useCognitionStream.ts (Next.js WebSocket Connection):
   - Replaced regex protocol translation with a robust `getWsUrl()` function:
     ```typescript
     const getWsUrl = (apiUrl: string): string => {
         try {
             const url = new URL(apiUrl);
             url.protocol = url.protocol === 'https:' ? 'wss:' : 'ws:';
             return url.toString().replace(/\/$/, '');
         } catch {
             return apiUrl.replace(/^http/, 'ws');
         }
     };
     ```
